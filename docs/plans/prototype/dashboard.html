E2E P95 时延
⏱ 端到端 95 分位时延
所有 session 的 duration_ms 排序后第 95 分位，代表"几乎全部用户"的体验下限：5% 的会话比这更慢。
P95 = sort(durations)[⌈N × 0.95⌉ − 1]

E2E 中位 P50
⏱ 端到端中位时延
典型用户体验：一半 session 比这快、一半比这慢。和 P95 配合看，差距越大说明长尾越严重。
P50 = median(durations)
